# Anywhere-Arcade

## About
Want a penny arcade on your web browser? Check out our app by clicking [here](https://anywhere-arcade.herokuapp.com/).

## Features
You can play:
* Tetris
* Sliding Puzzle

More games are comming soon.

## Development
Technologies used to build this app are listed below.
* Server：Heroku
* Back-end：Node.js
* Front-end：Vue.js
